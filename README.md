# prusa_exporter

prusa_exporter is implementation of "man-in-the-middle" for the [Buddy](https://github.com/prusa3d/Prusa-Firmware-Buddy?tab=readme-ov-file#buddy) based Prusa Research 3D printers. What it does is exposing metrics in Pormetheus format so it can be easily scraped.
